# Oqim

Cash flow analysis tool for small businesses, built as a static, client-side site —
no backend required.

## Deploy via GitHub Pages

1. Create a new repo on GitHub (e.g. `oqim`).
2. Upload `index.html` to the repo root (drag-and-drop via the GitHub web UI works fine,
   or `git add . && git commit -m "initial" && git push`).
3. Go to the repo's **Settings → Pages**.
4. Under "Build and deployment", set Source to **Deploy from a branch**, branch **main**,
   folder **/ (root)**. Save.
5. Wait ~1 minute. Your site will be live at:
   `https://YOUR-USERNAME.github.io/oqim/`

That's it — no environment variables, no functions, nothing else to configure.
